# Demo Thuật Toán Tối Ưu Hóa Bầy Đàn (PSO) - Bài Toán Hàm Sphere
Mã nguồn bao gồm cả thuật toán xử lý số liệu và mô phỏng hình ảnh động (animation) giúp trực quan hóa quá trình đàn hạt di chuyển về đích.

## Mô Tả Bài Toán 

Chúng ta cần tìm điểm đáy của một "cái bát" 3 chiều.
* **Hàm mục tiêu:** $f(x, y) = x^2 + y^2$
* **Không gian tìm kiếm:** $x, y \in [-10, 10]$
* **Mục tiêu:** Tìm giá trị $(x, y)$ sao cho $f(x, y)$ đạt giá trị nhỏ nhất (Min).
* **Đáp án lý thuyết:** Điểm cực tiểu toàn cục nằm tại tọa độ $(0, 0)$ với giá trị $f(0,0) = 0$.
